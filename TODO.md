# Books API ToDos

## `/books/`

- [X] `GET /api/books/{id}` — получение информации о книге.
- [X] `DELETE /api/books/{id}` — удаление книги.
- [ ] `PUT /api/books/{id}` — изменение или создание книги.
- [ ] `PATCH /api/books/{id}` — частичное изменение книги.
- [ ] Есть валидация ID автора.
